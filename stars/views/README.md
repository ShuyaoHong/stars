# Redesign of STARS View

This directory holds code for the prototyping of the new views in STARS.




## Controls

### Key-Bindings

 - `b` brushing
 - `l` linking
 - `r` reset
 - `t` travel
 - `z` zoom

### Mouse Events

### Zooming on

 - `Mouse Down` start lasso
 - `Mouse Down-Move` grow lasso
 - `Mouse Up` end lasso
